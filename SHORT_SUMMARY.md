# AdaptiveLearn AI: Short Summary

**Problem:** Online learning is static, lacks hands-free accessibility, and doesn't connect with real-world PDF materials.

**Solution:** A multi-modal AI Tutor (Gemini 1.5 Flash) with:
- **RAG Support:** Upload PDFs for document-specific chat/quizzes.
- **Voice UI:** Full Speech-to-Text and Text-to-Speech support.
- **Reports:** AI-driven progress analysis.

**Google Services Integration:**
- **Gemini 1.5 Flash & Vertex AI:** Core reasoning and model management.
- **text-embedding-004:** High-precision semantic search for RAG.
- **Cloud Run/Build:** Secure, scalable serverless deployment.
- **Speech/Translate:** Multi-modal & multi-lingual accessibility.

**Tech Challenges Fixed:**
- Updated SDK to **0.13.0** for Node 18 compatibility.
- Migrated to **Node.js backend** for secure API key management and XSS/Helmet protection.
- Built a robust **Vector indexing pipeline** for reliable PDF parsing.

**Conclusion:** A modern, secure, and Google-powered bridge between passive study and active cloud engineering.
